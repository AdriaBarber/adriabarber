Sobre mi

Mi nombre es Adrià Barber y soy diseñador de páginas web. Tengo experiencia en desarrollo de sitios web y un poco en c++.

Me aseguro de utilizar siempre las últimas tecnologías de desarrollo web: HTML5, CSS3. Si necesitas una página web moderna, funcional, que se adapte a todos los diferentes tipos de pantalla, te aseguro que llegaste al lugar correcto.

Puedes ponerte en contacto conmigo a través de Instagram, Gmail o completando el formulario de contacto que se encuentra al final de mi sitio web

Servicios

Diseño Web

Páginas web atractivas y funcionales con el objetivo de satisfacer todas tus expectativas desde qualquier dispositivo. La importancia del diseño es fundamental para tener éxito y atraer a tus visitantes. También lo es la velocidad de carga del sitio web, deben ser rápidos para que el visitante tenga la mejor experiencia.

Mantenimiento

Puedes contactarme si necesitas actualizar tu página web, agregar secciones o cualquier cambio que necesites. Esto es perfecto para los sitios web de pequeñas empresas, donde no se necesita un diseñador web a tiempo completo, sino solo en distintas ocasiones.

Cursos

Cursos variados para aprender a programar páginas Web o aprender a diseñar (logos, banners etc...).

SEO

Al crear sitios web tengo muy en cuenta la optimización de motores de búsqueda (SEO). Lo que permite que tu sitio tenga una clasificación más alta en los resultados de Google y así generar más tráfico a la web.
