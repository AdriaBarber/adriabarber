Sobre mi

¡Hola! Soy Adrià Barber, diseñador Web. Tengo experiencia en desarrollo de sitios web para pequeñas y medianas empresas, hasta tiendas de compras online. Me aseguro de utilizar siempre las últimas tecnologías de desarrollo web: HTML5, CSS3, jQuery, WordPress. Si necesitas una web moderna, funcional y que se adapte a todos los diferentes tipos de pantallas, te aseguro que has venido al lugar indicado.

Desarrollo Web

Páginas web atractivas y funcionales con el objetivo de satisfacer todas tus espectativas. La importancia del diseño es fundamental para tener éxito y atraer a tus visitantes. También lo es la velocidad de carga del sitio web, deben ser rápidos para que el visitante tenga la mejor experiencia

Diseño Responsive

Hoy en día las páginas web se ven en multitud de dispositivos como teléfonos movil, tablets, PCs etc. La tecnología móvil es el presente y el futuro. Para eso es importante y me aseguro de que tu sitio web sea compatible con todos los tamaños de pantalla de esta época moderna

Mantenimiento

Puedes contactarme si necesitas actualizar tu página web, agregar nuevas secciones o cualquier cambio que necesites. Esto es perfecto para los sitios web de pequeñas empresas, donde no se necesita un diseñador de páginas web a tiempo completo, sino solo en distintas ocasiones
